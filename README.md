# EduTech-Task13-AB-Testing
# Task 13 - A/B Testing & Hypothesis Testing

## Tools Used
Python, Pandas, Scipy, Statsmodels, Matplotlib, Seaborn

## Dataset
A/B Testing Dataset (Kaggle)
Link: https://www.kaggle.com/datasets/zhangluyuan/ab-testing

## Steps Performed
1. Split data into Group A (control) and Group B (treatment)
2. Calculated conversion rates for both groups
3. Set up Null and Alternative Hypothesis
4. Performed Independent T-test
5. Calculated 95% Confidence Intervals
6. Visualized conversion rate comparison and distribution

## Hypothesis
- H0: No difference between Group A and Group B
- H1: Significant difference exists between groups

## Results
- P-value determines whether to reject or accept Null Hypothesis
- Confidence Intervals show range of true conversion rate

## Interview Answers
**What is A/B testing?**
A/B testing ek statistical method hai jisme do versions (A and B) ko compare karte hain yeh dekhne ke liye ki kaunsa better perform karta hai. Jaise website ke 2 designs compare karna user conversion ke liye.

**Confidence interval?**
Confidence Interval ek range hota hai jisme hum confident hain ki true population value aayegi. 95% Confidence Interval ka matlab hai 95% confidence ki actual value is range ke andar hai.
